## Compilación con ADead-BIB

```bash
# Ejemplo: compilar zlib
tar xzf zlib-1.3.1.tar.gz
cd zlib-1.3.1
adB cc -c *.c -O2
ar rcs libz.a *.o

# Ejemplo: compilar sqlite
tar xzf sqlite-autoconf-3440200.tar.gz
cd sqlite-autoconf-3440200
adB cc -c sqlite3.c -O2 -DSQLITE_OMIT_LOAD_EXTENSION
ar rcs libsqlite3.a sqlite3.o
```
